# Game Typography System

## Text Styles (System Fonts Only)

### Small UI
- **Font Family**: System UI (Cambay as fallback)
- **Size**: 12px
- **Usage**: Secondary labels, metadata, annotations
- **No custom styling applied**

### HUD Numeric
- **Font Family**: Monospace
- **Size**: 14px
- **Usage**: Counters, scores, numerical data
- **No custom styling applied**

### Label / Annotation
- **Font Family**: System UI (Cambay as fallback)
- **Size**: 11px
- **Usage**: Helper text, instructions, guides
- **No custom styling applied**

## Typography Rules

- No display fonts
- No stylization or decorative treatments
- System fonts ensure readability and performance
- Text should communicate clearly without visual noise
- Hierarchy through size and weight only, not color or effects

## Implementation Notes

These are style definitions only. No visual treatment has been applied to any game components yet.
